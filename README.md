# Glyph Flow

Animated typographic fields. A stable micro-type texture is revealed by a moving organic field, producing large flowing masses, eroded voids, and sparse dust. Deterministic for the same seed and timeline position.

**Live:** https://natochi.cv/tools/glyph-flow/

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build          # outputs to dist/
npm run verify:final   # typecheck, tests, build, browser tests
```

Built with React, Vite and TypeScript. See `specs/` for the product spec.
